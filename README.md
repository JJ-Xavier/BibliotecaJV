# 📚 Sistema de Biblioteca - UCB

![Java](https://img.shields.io/badge/java-%23ED8B00.svg?style=for-the-badge&logo=openjdk&logoColor=white)

## 📖 Sobre o Projeto
Este sistema de gerenciamento de biblioteca foi desenvolvido como parte da disciplina de **Programação Orientada a Objetos** na **Universidade Católica de Brasília (UCB)**.
O objetivo principal foi aplicar os pilares da POO em um cenário real de gestão de acervo e empréstimos.

## 🚀 Funcionalidades Concluídas
- **Cadastro de Acervo:** Gerenciamento completo da classe `Livro`.
- **Controle de Usuários:** Registro e identificação de leitores.
- **Fluxo de Empréstimo:** Lógica de negócio para retirada e devolução de livros.
- **Validação de Disponibilidade:** Sistema que impede o empréstimo de livros que já estão fora do acervo.

## 🛠️ Tecnologias e Ferramentas
- **Linguagem:** Java
- **IDE Utilizada:** IntelliJ IDEA (Migrado do Eclipse)
- **Versionamento:** Git & GitHub
- **Paradigma:** Orientação a Objetos (Encapsulamento, Herança e Polimorfismo)

## 🏗️ Estrutura do Código
O projeto está organizado nas seguintes classes principais:
1. `Main`: Ponto de entrada que executa o menu interativo.
2. `Livro`: Atributos da obra (Título, Autor, ISBN).
3. `Biblioteca`: Gerencia a lista de livros e as operações de busca.
4. `Emprestimo`: Controla as transações entre o acervo e o usuário.

## 🏁 Como Rodar o Projeto
Como o projeto foi desenvolvido utilizando o **IntelliJ IDEA**, basta seguir estes passos:
1. Importe a pasta do projeto no IntelliJ.
2. Certifique-se de que o **JDK** (versão 17 ou superior) está configurado corretamente nas *Project Structure settings*.
3. Execute a classe `Main.java`.
